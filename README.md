# Планировщик шашлыка

Статический сайт-помощник для подготовки шашлыка и маринада. Пользователь выбирает мясо, вес и дату готовки, а приложение считает соль, специи, основу маринада, подсказывает время маринования и собирает список покупок.

## Возможности

- Выбор мяса: свинина, курица, говядина, баранина, рыба.
- Расчет ингредиентов по весу мяса.
- Быстрые сценарии: сегодня, завтра, через 3 дня.
- Подсказка, когда начинать мариновать.
- Список покупок с мясом, солью, луком, маслом, основой маринада и специями.
- Стиль маринада `Собственный` с модулем специй.
- Чекбоксы для выбора специй и автоматический пересчет граммовки.
- Кнопка копирования готового плана и списка покупок.

## Модуль специй

В модуле есть практичный набор специй для шашлыка и готовки:

- перцы и паприка;
- восточная база;
- сушеные травы;
- сухие овощные специи;
- готовые смеси;
- ароматические акценты.

Для собственного маринада выбранные специи добавляются в список покупок отдельными строками с расчетом граммов на выбранный вес мяса.

## Как открыть

Проект не требует сборки и зависимостей.

1. Скачайте или клонируйте репозиторий.
2. Откройте `index.html` в браузере.
